Programming Tutorials

A proficient developer should know several programming languages. Using these tutorials you can survey top programming languages. We cover basic syntax and semantics for a quick start. After you become familiar with fundamental concepts of a language you can decide to learn another or deep dive to master it using the reference manual.

Prerequisite

Learning a programming language is expensive investment of time and effort. We have made a simple quiz that you can try before you start learning a programming language. If you score less than 80 points you should take our Software Engineering class to learn the fundamentals. You need a Google account to take this quiz.

Programming Quiz

Repeating: You can take this quiz only once. If you fail, you can ask for suppoort on Discord in the general chat room. An instructor can review your answers and reset your score so you can try a second time. If you fail second time don't be descouraged, start with most easy programming language.
