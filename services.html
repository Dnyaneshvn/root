Servers

To achieve performance at low cost we use our own custom made micro-servers. We have setup two Nvidia rigs. These rigs have 2 video cards GTX 780Ti 8Gb RAM and Intel Core 4 processor. They are identical and they use similar software. We host two databases with GPU accelerators and SSD metadata.

For high performance computing we use 2 custom made Linux GPU rigs. These rigs have 4xGPU RX480 8Gb AMD video cards working in stand/by mode. Each machine have access to large capacity network storage for data and logs. We use these two machines for running GPU intensive computation jobs in parallel.

Your build will be scheduled every day on same hour. We interlace projects so the build time is distinct for each project. After build we run a very short test called "unit test" if this is available. We push the release artifacts on-line for download. You will be able to download most recent release using FTP or HTTP protocol.

For Linux build jobs we use a cubic black machine named Borg. This machine has large air flow fans one huge video card and two secondary WD enterprise 1TB HDD. We use PowerColor Red Dragon AMD Radeon™ RX 6800 Gaming Graphics Card with 16GB and NVMe WD 500Gb accelerator card. This is our fastest machine. It run in stand-by mode and wakes up to run the builds then goes back to sleep automatically to save power.

Testing

We enable creation of test automation services. This is done on our back-end servers using test scripts. We set-up several testing jobs customized for the specifics of your project:dry-run, smoke-test, nightly-test and weekly-test can be scheduled. We publish test logs for you to download and investigate. Not all projects require test automation.

Robot

We enable creation of fast computation agents named robots that can be run on schedule to build your project every week. A robot is a script that can be executed on a multi-core CPU or powerful GPU rig in parallel. We set-up the job by your specification, and schedule the job to run repeatedly one time/week for a total of maximum 52h/year. This robot can use data from a database to generate HTML pages or can download something from the internet on our server and run a test in parallel on powerful 5GPU. The robot can be scheduled between 6:00AM to 6:PM CDT, for any work day of the week, and can run one time/week.You must make the job as short as possible maximum 1h/week. Price is fix 20$/year not negociable.

Note: You can't mine crypto using this robot, but you can build a prototype for High Performance Computation jobs. We reserve service time exclusive for 1 job at a time so it will not run in parallel with any other jobs. This way you can verify the time spent for your job run and optimize the process. We have a capacity of 2 rigs.

Backup

Our most precious service is the backup. We perform a periodic backup for test data fixtures on internal HDD automatically every night. We save your code on external media every weekend. The code need less backup since it is safely stored on Mirror RAID. You can also do backup for data stored on the internet. This is a convenient way to protect your external projects or data from hacker attacks. It is better to be proactive than reactive.

We use a dedicated backup machine. This is a workstation with blue-ray, DVD writer, USB3 and memory card slots. It has large capacity internal RAID, WD Raptor OS disk (no SSD), redundant high capacity storage and external HDD slots. This machine is very silent sturdy and heavy. It has water cooling system for micro processor, two video cards and licensed antivirus. This is our most robust build.

Archive

We offer a long term archive for your project on HDD, DVD, BD or SD Card. Maxim size is 4/25/32/250 Gb depending on media size. We create this archive on demand one single time/year. We store the archive safely in a vault, protected against sun-light, dust, heat and humidity. If you have an archive we can restore from it and provide a link for download on FTP storage. We encrypt your project archive with a secret code that you will need in order to extract the archive content. We will communicate this code using encrypted messages directly on your mobile phone.
