We receive payments on-line using PayPall. This is not a subscription that goes automatically. We know you hate that. We send you an invoice/year. You can pay this invoice in multiple payments until you cover the entire cost. We let you decide how much to pay and how often.

Infrastructure

We run a tiny data center in Chicago. We do not offer a dedicated IP or domain name but we can offer one subdomain @sagecode.net for each member. We can build and upload your project for visitors or download. Our infrastructure is not glamorous but efficient and has enough power for software development, data transformation backup and deploy.

Software Support

We do not host VPS. We use shared server resources. Our servers are updated regularly with the latest software but services are running on bare bone machine. We install most stabile versions of compilers. You can contact us on Discord if your project require special compilers or libraries. The support is direct on chat using DM.

Servers

To achieve performance at low cost we use our own custom made micro-servers. We have setup two Nvidia rigs. These rigs have 2 video cards GTX 780Ti 8Gb RAM and Intel Core 4 processor. They are identical and they use similar software. We host databases with GPU accelerators and SSD metadata.

For high performance computing we use 2 custom made Linux GPU rigs. These rigs have 4xGPU RX480 8Gb AMD video cards working in stand/by mode. Each machine have access to large capacity network storage for data and logs. We use these two machines for running GPU intensive computation jobs in parallel.

Limits

We have limited bandwidth and disk space so we must cap the resources you can use. A regular portfolio project is usually small and has no more than 5 developers in a team. A decent portfolio project should never grow more than 2Gb. If your project is larger we will ask you to clean-up the garbage and shrink it down. A large project is usually a bad portfolio project.

Note: At this time we can't scale up due to grid power limitation but our capacity is currently 10% occupied so we are confident we can host at least 100 projects with plenty of reserve remaining. In the future we will install solar panels for lowering our electricity bills and expand our capacity.

Robot

We enable creation of fast computation agents named robots that can be run on schedule to build your project. A robot is a script that can be executed on a multi-core CPU or powerful GPU rig in parallel. We set-up the job by your specification, and schedule the job to run repeatedly one time/week for a total of maximum 52h/year. This robot can use data from a database to generate HTML pages or can download something from the internet on our server in parallel and run a test. The robot can start any day of the week and can run one time for maxim 1h/week then it is stopped automatically even if it does not finish the job. You must make the job as short as possible.

Note: You can't mine Ethereum or any other crypto using this robot, but you can build a prototype for High Performance Computation job. We reserve service time exclusive for 1 job at a time so it will not run in parallel with any other jobs. This way you can verify the time spent for your job run and optimize the process.

Backup

Our most precious service is the backup. We perform a periodic backup for test data on internal HDD automatically every night. We save your code on external media every weekend. The code need less backup since it is safely stored on Mirror RAID. You can also do backup for data stored on the internet. This is a convenient way to protect your external projects or data from hacker attacks. We can download data in parallel using high speed internet.

We use a dedicated backup machine. This is a workstation with blue-ray, DVD writer, USB3 and memory card slots. It has large capacity internal RAID, WD Raptor OS disk (no SSD), redundant high capacity storage and smaller capacity external HDD. This machine is very silent sturdy and heavy. It has water cooling system for micro processor, two video cards and licensed antivirus. This is our most robust build.

Archive

We offer a long term archive for your project on HDD, DVD, BD or SD Card. Maxim size is 4/25/32/250 Gb depending on media size. We create this archive on demand one single time/year. We store the archive safely in a vault, protected against sun-light, dust, heat and humidity. If you have an archive we can restore from it and provide a link for download on FTP storage. We encrypt your project archive with a secret code that you will need in order to extract the archive content. We will communicate this code using encrypted messages directly on your mobile phone.
