Price

With all options selected, the maximum price will be 48$/year. A regular project needs minimum 3 services in total value of 10$/year. We do not accept orders with less value. If you can't afford it you should host your project on GitHub or other website that offer free hosting services.

Payments

We receive payments on-line using PayPall. This is not a subscription plan that goes automatically. Instead you pay as you go. You can request an invoice on Discord if you need one for your business. Payment is optional. You pay only if you are satisfied with our services.

Infrastructure

We run our server in Chicago office. Our infrastructure is not glamorous but efficient and has enough power for software development, data transformation jobs, backup, archive and software delivery using upload. We have in production two custom made Linux GPU mining rigs, two database micro servers, one Linux automation server and one Windows backup workstation.

Software Support

We use shared server resources not VPS. Our servers are updated regularly with the latest software but services are running on bare bone machine. We install most stabile versions of compilers. You can contact us on Discord if your project require special compilers or libraries. The support is direct on chat using DM.

Limits

We have limited bandwidth and disk space so we must cap the resources. A regular portfolio project is usually small and has no more than 5 developers. A decent portfolio project should never grow more than 2Gb. If your project is larger we will ask you to prune it. A large project is usually a bad project.

Note: At this time we can't scale but our load is currently under 1% so we are confident we can host about 100 projects. In the future we will install solar panels for lowering the electricity bills and expand our capacity.

Service Details

Exclusive: We accept projects only from active members that have a testimonial page. You can order a testimonial after you finish the programming tutorial and pass the prep quiz. We host testimonial pages free of charge. On the testimonial page we list your projects and provide links to open/view/download or run your project for demo.

Git Repository

We offer private Git repository to do version control. This service has a cap limit of 2Gb/project and is available 24/7. This repository will be hosted on our domain server in Los Angeles. You can order this repository to improve security of your development pipeline.

FTP Storage

We offer FTP file transfer storage to upload test data and download project released files or computation results. This service has a cap limit of 3Gb/project and is available 24/7. This service is hosted in Los Angeles on our domain server.

Database

You can build a database application or data pipeline using our infrastructure. We can set-up a small database that you can use to create back-end for a small business. We can assist with creation of custom structure and required data pipeline using automation services.

Build

Your build will be scheduled every day on the same hour. We interlace projects so the build jobs are using full server capaciti for each project. After build we run a very short test called "unit test" when this is available. After test, we archive and push the release artifacts on-line for download. You will be able to download most recent release packages using FTP or HTTP protocol.

Testing

We enable creation of functional end2end test. This is done using Jenkins test automation platform. We set-up several testing jobs customized for the specifics of your project: dry-run, smoke-test, nightly-test and weekly-test can be scheduled. We publish test logs for you to download and investigate. Not all projects require test automation.

Robot

We enable creation of fast computation agents that can run on schedule to build your project every week. A robot is a script that can be executed on a multi-core CPU or a powerful GPU rig in parallel. We schedule the robots to run 1h/week for a total of maximum 52h/year.

Note: We reserve service time exclusive for 1 job at a time so it will not run in parallel with any other jobs. This way you can verify the time spent for your job to find performance bottelnecks and improve your algorithms.

Backup

We perform a periodic backup on internal HDD automatically every day. You can backup test data and source code. This is a convenient way to protect your projects or data from hacker attacks. Our backup can be used to download data from the cloud before backup.

We use a dedicated backup machine. This is a workstation with blue-ray, DVD writer, USB3 and memory card slots. It has large capacity internal RAID, WD Raptor OS disk (no SSD), redundant high capacity storage and external HDD slots. This machine is very silent sturdy and heavy. This is our most robust build.

Archive

We offer a long term archive for your project on HDD, DVD, BD or SD Card. We create this archive on demand one single time/year. We store the archives safely in a vault, protected against sun-light, dust, heat and humidity. If you have an archive we can restore from it and provide a link for download using FTP.

Trial Period

You can use our services free of charge on a negociable period of time depending on your project demands. If the project is very small we may not charge you at all. If a project grow above the limits we will not shut it down but we may negociate an overcharge.
