Engineering Tutorials

Software engineering is a branch of computer science that deals with the design, implementation, and maintenance of complex computer programs. It consists of systematic application of engineering methods to software production.

Audience

This course is for new developers that have little or no knowledge about computer programming. We use Python for occasional examples but this is not a Python class. In this class we will prepare you with basic know-how that you can use later to learn any programming language.

Curriculum

Next lectures will teach you how to create higher quality software and avoid catastrophic defects. We have selected the most fundamental topics that will extend your vision about computer programming. After you study these you will be able to select the right technology and establish a development process for your future projects.
